# UTI Detection Chatbot - Deployment Package
    
This zip contains everything needed to deploy your AI-powered UTI detection chatbot.
    
Files included:
- uti_chatbot_app.py (main application)
- requirements.txt (dependencies)
- models/ (trained AI model and preprocessing files)
    
Deployment:
1. Upload all files to GitHub
2. Deploy on Streamlit Cloud
3. Share your live app URL
